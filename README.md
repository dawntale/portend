### Installation

- Clone / Download this repo
- Rename .env.example to .env
- Create database and insert your dbname, dbuser, and dbpass in the .env file
- Run
 - `composser update`
 - `php artisan migrate`
 - `php artisan db:seed`
- Dashboard `example.dev/dashboard`
